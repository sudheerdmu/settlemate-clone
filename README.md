# Settlemate Clone

A React Native Expo project for finding class action settlements.

## Project Structure

```
.
├── ai-logs/          # AI session logs and prompts
├── screenshots/      # Project screenshots and demos
├── src/              # Source code (to be created)
└── package.json      # Dependencies
```

## Getting Started

```bash
# Install dependencies
npm install

# Start Expo development server
npx expo start
```

## AI Development Log

See `ai-logs/` directory for detailed session logs of AI-assisted development.
